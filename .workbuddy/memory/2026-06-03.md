## 前后端连接稳定性改进

**问题**: 前端频繁出现"后端未连接"错误，原因是后端接口变化导致连接失败

**解决方案**:
1. 创建统一的 API 客户端封装 (`frontend/src/utils/api-client.js`)
   - 自动健康检查轮询 (5 秒间隔)
   - 智能指数退避重连 (1 秒 → 30 秒，最大 10 次尝试)
   - 请求队列缓存 (10 分钟超时，最多重试 3 次)
   - 统一错误处理和回调

2. 重构 `App.vue` 使用新的 API 客户端
   - 替换所有 `fetch(API_BASE + ...)` 调用为 `apiClient.get/post(...)`
   - 添加连接状态监听
   - 在 `mounted` 和 `beforeUnmount` 中管理健康检查生命周期

3. 文档化改进 (`docs/CONNECTION_IMPROVEMENTS.md`)
   - 技术细节说明

   - 测试建议
