## "用 AI 增强提示"功能实现

**问题**: 用户报告"用 AI 增强提示"按钮点击后输入框文本无变化

**根本原因**: 该功能在项目中有**完全未实现**（前端无按钮和逻辑，后端无 API 接口）

**解决方案**: 从头实现完整功能

### 后端实现 (`backend/routers/event.py`)

1. **添加请求模型** (第 18-20 行):
```python
class PromptEnhanceRequest(BaseModel):
    prompt: str  # 用户输入的原始提示词
```

2. **添加 API 接口** (第 328-411 行) `enhance_prompt_api`:
   - 接收用户简短关键词（如"疫情"）
   - 构建系统提示词引导 LLM 扩写为 200-400 字详细事件描述
   - 包含时间、地点、人物、发展过程、社会影响等要素
   - 使用 `messages` 列表格式调用 LLM（修正了最初错误的 `system` 参数）
   - 内容长度校验：如果生成内容过短（<50 字符），自动重试一次
   - 返回 `enhanced_prompt` 和 `original_prompt`

### 前端实现 (`frontend/src/App.vue`)

1. **添加状态变量** (在 `data()` 中):
   - `enhancing`: 加载状态
   - `enhanceError`: 错误信息
   - `enhanceSuccess`: 成功状态

2. **添加 UI 组件**:
   - "用 AI 增强提示"按钮（禁用状态随加载变化）
   - 加载提示、错误提示、成功提示

3. **实现 `enhancePrompt` 方法**:
   - 检查输入框内容非空
   - 调用后端 `/api/event/enhance-prompt` API
   - 成功后更新 `airdropContent` 输入框
   - 自动聚焦并高亮显示增强后的内容
