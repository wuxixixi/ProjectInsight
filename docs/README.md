# 觉测·洞鉴 - 信息茧房推演系统

> 多智能体舆论认知干预沙盘

**在线演示**: http://101.34.62.149

## 项目简介

觉测·洞鉴（ProjectInsight）是一个可视化模拟系统，用于研究算法推荐与官方辟谣对群体观点演化的影响。该系统支持两种推演模式：

1. **数学模型模式**：基于社会心理学理论的快速推演，适合参数探索和大规模模拟
2. **LLM 驱动模式**：基于大语言模型的智能体决策，模拟真实人类的认知行为和决策过程

## 核心功能

### 推演模式

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| 数学模型 | 快速、可复现、适合参数敏感度分析 | 教学演示、大规模模拟 |
| LLM 驱动 | 模拟真实人类认知行为、决策过程可解释 | 深度研究、智库专报生成 |

### 主要特性

- **算法茧房效应模拟**：可调节的推荐算法强化程度，观察观点极化过程
- **双层社交网络**：模拟公域（微博式）与私域（微信式）信息传播的差异
- **知识图谱解析**：将新闻事件解析为实体-关系图谱，Agent 基于结构化知识理解事件
- **官方辟谣机制**：可设置延迟时间，研究辟谣时机对舆论的影响
- **智能体透视**：点击网络节点查看 Agent 的完整决策过程和人设信息
- **智库专报生成**：基于 LLM 自动生成专业的舆情分析报告

### 实时可视化

- **观点分布直方图**：红色=相信谣言，橙色=中立，绿色=相信真相
- **信息传播网络拓扑**：节点颜色代表观点，大小代表影响力
- **舆论演化趋势曲线**：追踪谣言传播率、真相接受率、极化指数等指标

## 快速开始

### 环境准备

```bash
# 创建虚拟环境 (推荐)
conda create -n info-cocoon python=3.10
conda activate info-cocoon

# 或使用 venv
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

### 安装依赖

```bash
# 后端依赖
pip install -r requirements.txt

# 前端依赖
cd frontend
npm install
```

### 启动服务

**终端 1 - 后端服务：**
```bash
uvicorn backend.app:app --reload --port 8000
```

**终端 2 - 前端开发服务器：**
```bash
cd frontend
npm run dev
```

访问 http://localhost:3000 打开界面。

## 参数说明

| 参数 | 范围 | 说明 |
|------|------|------|
| 推演模式 | LLM驱动/数学模型 | LLM模式模拟真实人类决策，数学模型快速推演 |
| 算法茧房强度 | 0 ~ 1 | 越高越容易强化用户既有观点 |
| 官方辟谣延迟 | 0 ~ 30 步 | 谣言传播多久后发布辟谣信息 |
| 初始谣言传播率 | 10% ~ 60% | 初始相信谣言的人群比例 |
| Agent数量 | 50 ~ 500 | 模拟群体规模 |
| 社交网络类型 | 小世界/无标度/随机 | 网络拓扑结构 |
| 双层网络 | 开启/关闭 | 模拟公域+私域信息传播 |

## 项目结构

```
ProjectInsight/
├── backend/                 # Python 后端
│   ├── app.py              # FastAPI 主入口
│   ├── simulation/         # 推演核心逻辑
│   │   ├── engine.py       # 数学模型引擎
│   │   ├── engine_dual.py  # 双层网络引擎
│   │   ├── agents.py       # 数学模型智能体
│   │   ├── llm_agents_dual.py # LLM 驱动智能体(含知识图谱)
│   │   ├── graph_parser_agent.py # 知识图谱解析器
│   │   └── analyst_agent.py # 智库分析师 Agent
│   ├── llm/                # LLM 客户端
│   └── models/             # 数据模型
├── frontend/               # Vue 3 前端
├── docs/                   # 项目文档
├── reports/                # 生成的报告
└── tests/                  # 测试套件
```

## 工作流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         信息茧房推演系统工作流程                            │
└─────────────────────────────────────────────────────────────────────────┘

1. 输入新闻/事件
        ↓
2. 知识图谱解析 (LLM)
   ┌─────────────────────────────────────────┐
   │  实体提取: 人物、组织、地点、事件         │
   │  关系抽取: 关联、影响、参与               │
   │  语义摘要: 事件概述                       │
   └─────────────────────────────────────────┘
        ↓
3. Agent 决策 (LLM)
   ┌─────────────────────────────────────────┐
   │  输入: 邻居观点 + 知识图谱 + 辟谣信息      │
   │  输出: 新观点 + 情绪 + 行动 + 评论        │
   └─────────────────────────────────────────┘
        ↓
4. 观点演化 + 可视化
```
