# 觉测·洞鉴 - 信息茧房推演系统

> 多智能体舆论认知干预沙盘

**上海社会科学院** | **在线演示**: http://101.34.62.149

## 项目简介

觉测·洞鉴（ProjectInsight）是由上海社会科学院开发的可视化模拟系统，用于研究算法推荐与官方辟谣对群体观点演化的影响。该系统支持两种推演模式：

1. **沙盘推演模式**：参数驱动，研究传播机制，适合假设验证
2. **新闻推演模式**：真实分布锚定，预测现实演进，支持干预决策

## 核心功能

### 推演模式

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| 沙盘推演 | 参数完全可控、快速复现 | 教学演示、参数敏感度分析 |
| 新闻推演 | 真实分布锚定、预测区间 | 舆情预警、干预时机决策 |
| 数学模型 | 毫秒级响应 | 大规模模拟 |
| LLM驱动 | 模拟真实人类决策 | 深度研究、智库专报 |

### 主要特性

- **双模式推演**：沙盘模式探索传播机制，新闻模式预测现实演进
- **真实分布锚定**：支持上传真实舆情数据初始化Agent观点分布
- **预测区间**：输出置信区间而非单值预测，科学评估不确定性
- **干预时机建议**：自动分析最佳干预时机和策略
- **双层社交网络**：模拟公域（微博式）与私域（微信式）信息传播差异
- **知识图谱解析**：将新闻事件解析为实体-关系图谱，Agent基于结构化知识决策
- **快速注入**：支持跳过图谱解析，秒级响应事件注入
- **智能体透视**：点击网络节点查看Agent的完整决策过程和人设信息
- **智库专报生成**：基于LLM自动生成专业的舆情分析报告

### 实时可视化

- **观点分布直方图**：红色=相信谣言，橙色=中立，绿色=相信真相
- **信息传播网络拓扑**：节点颜色代表观点，大小代表影响力
- **舆论演化趋势曲线**：追踪谣言传播率、真相接受率、极化指数等指标
- **预测轨迹展示**：乐观/悲观/期望三种场景演化预测

## 快速开始

### 环境准备

```bash
# 创建虚拟环境 (推荐)
conda create -n info-cocoon python=3.10
conda activate info-cocoon

# 或使用 venv
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

### 安装依赖

```bash
# 后端依赖
pip install -r requirements.txt

# 前端依赖
cd frontend
npm install
```

### 启动服务

**终端 1 - 后端服务：**
```bash
uvicorn backend.app:app --reload --port 8000
```

**终端 2 - 前端开发服务器：**
```bash
cd frontend
npm run dev
```

访问 http://localhost:3000 打开界面。

## 参数说明

| 参数 | 范围 | 说明 |
|------|------|------|
| 推演模式 | 沙盘/新闻 | 沙盘模式参数驱动，新闻模式真实锚定 |
| 推演引擎 | LLM驱动/数学模型 | LLM模式模拟真实人类决策，数学模型快速推演 |
| 算法茧房强度 | 0 ~ 1 | 越高越容易强化用户既有观点 |
| 官方辟谣延迟 | 0 ~ 30 步 | 谣言传播多久后发布辟谣信息 |
| 初始谣言传播率 | 10% ~ 60% | 初始相信谣言的人群比例 |
| Agent数量 | 50 ~ 500 | 模拟群体规模 |
| 社交网络类型 | 小世界/无标度/随机 | 网络拓扑结构 |
| 双层网络 | 开启/关闭 | 模拟公域+私域信息传播 |

## 事件注入

系统支持两种事件注入方式：

### 完整解析模式（推荐首次注入）
- 调用LLM解析知识图谱
- 提取实体、关系、情感
- 适用于深度分析
- 响应时间：10-60秒

### 快速注入模式
- 跳过图谱解析
- 秒级响应
- 适用于推演中快速注入
- 响应时间：1-2秒

## 工作流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         信息茧房推演系统工作流程                            │
└─────────────────────────────────────────────────────────────────────────┘

1. 选择推演模式
   ├── 沙盘模式：设置参数，探索传播机制
   └── 新闻模式：输入真实舆情分布，预测现实演进

2. 注入事件（可选）
   ├── 完整解析：LLM解析知识图谱（10-60秒）
   └── 快速注入：跳过解析，秒级响应

3. 开始推演
   ├── 数学模型：毫秒级响应
   └── LLM驱动：模拟真实人类决策

4. 实时监控
   ├── 观点分布演化
   ├── 预测轨迹展示
   └── 风险预警提示

5. 干预决策（新闻模式）
   ├── 最佳干预时机建议
   ├── 干预策略推荐
   └── 预期效果评估

6. 生成报告
   └── 智库专报：宏观趋势+微观切片+政策建议
```
