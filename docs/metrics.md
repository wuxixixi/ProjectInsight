# 指标解释

## 概述

系统通过多个指标量化舆论演化过程，帮助研究者理解信息传播和观点演化的规律。新闻模式下还提供预测区间和风险预警指标。

> **命名说明**: 经过语义重构，指标命名已从"谣言/辟谣"抽象为"负面信念/权威回应"，但核心计算逻辑不变。API同时返回新旧两种字段名以确保向后兼容。

## 核心指标

### 1. 负面信念率 (Negative Belief Rate)

> 旧名: `rumor_spread_rate`（误信率）

**定义**: 当前误信负面信念的人群比例

**计算公式**:
```
negative_belief_rate = count(opinion < OPINION_THRESHOLD_NEGATIVE) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 越高表示负面信念传播越广泛
- 权威回应后应逐渐下降

**可视化**: 红色区域面积占比

---

### 2. 正面信念率 (Positive Belief Rate)

> 旧名: `truth_acceptance_rate`（正确认知率）

**定义**: 当前持有正确认知的人群比例

**计算公式**:
```
positive_belief_rate = count(opinion > OPINION_THRESHOLD_POSITIVE) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 越高表示正面信念被越多的人接受
- 权威回应后应逐渐上升

**可视化**: 绿色区域面积占比

---

### 3. 平均观点 (Average Opinion)

**定义**: 群体观点的平均值

**计算公式**:
```
avg_opinion = sum(opinion_i) / N
```

**解读**:
- 范围: -1 ~ +1
- 负值: 群体倾向负面信念
- 正值: 群体倾向正面信念
- 0: 群体观点中立

**可视化**: 观点分布直方图的"重心"

---

### 4. 极化指数 (Polarization Index)

**定义**: 群体观点的分歧程度

**计算公式**:
```
polarization = sqrt(mean(opinion_i^2))
```

或使用标准差变体:
```
polarization = std(opinion)
```

**解读**:
- 范围: 0 ~ 1
- 0: 群体观点完全一致
- 1: 群体观点两极分化（完全负面信念 vs 完全正面信念）

**高极化场景**:
- 双方互相不信任
- 信息茧房效应强
- 缺乏共同信息基础

---

### 5. 沉默率 (Silence Rate)

**定义**: 不表达观点的人群比例

**计算公式**:
```
silence_rate = count(is_silent == True) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 反映"沉默的螺旋"效应
- 高沉默率可能导致极端观点主导舆论

## 双层网络指标

当启用双层网络模式时，额外提供以下指标：

### 6. 公域负面信念率 (Public Negative Rate)

> 旧名: `public_rumor_rate`

**定义**: 公域网络中误信负面信念的用户比例

**计算范围**: 仅公域网络中的节点

### 7. 公域正面信念率 (Public Positive Rate)

> 旧名: `public_truth_rate`

**定义**: 公域网络中持有正面信念的用户比例

### 8. 私域负面信念率 (Private Negative Rate)

> 旧名: `private_rumor_rate`

**定义**: 私域社群中误信负面信念的用户比例

### 9. 私域正面信念率 (Private Positive Rate)

> 旧名: `private_truth_rate`

**定义**: 私域社群中持有正面信念的用户比例

### 10. 社群数量 (Number of Communities)

**定义**: 私域网络中独立社群的数量

**说明**: 反映信息在私域中的隔离程度

### 11. 大V数量 (Number of Influencers)

**定义**: 公域网络中超级节点的数量

**说明**: 高影响力用户，可左右舆论走向

## 预测指标（新闻模式）

### 12. 预测区间 (Prediction Interval)

**定义**: 未来某指标的可能范围

**结构**:
```json
{
  "expected": 0.52,      // 期望值（最可能结果）
  "optimistic": 0.38,    // 乐观场景（下界）
  "pessimistic": 0.66,   // 悲观场景（上界）
  "confidence": 0.95     // 置信度
}
```

**解读**:
- `expected`: 基于当前趋势的最可能结果
- `optimistic`: 最好情况下可能达到的值
- `pessimistic`: 最坏情况下可能达到的值
- `confidence`: 预测的可信程度（通常为95%）

**可视化**: 预测轨迹图中的三条曲线

---

### 13. 风险等级 (Risk Level)

**定义**: 综合评估当前舆情风险

**风险等级含义**:

| 等级 | 触发条件 | 含义 | 建议行动 |
|------|----------|------|----------|
| critical | 误信率>50% 或 极化>0.8 | 危急 | 立即干预，启动应急预案 |
| high | 误信率>35% 或 极化>0.6 | 高危 | 高度关注，准备干预 |
| medium | 误信率>20% 或 极化>0.4 | 中等 | 持续监控，评估态势 |
| low | 其他 | 低风险 | 常规观察 |

**风险规则命名**:

| 规则名 | 监控指标 | 阈值 |
|--------|----------|------|
| `negative_critical` | negative_belief_rate | > 0.7 |
| `negative_high` | negative_belief_rate | 0.5 ~ 0.7 |
| `negative_rising` | negative_belief_rate | 连续上升趋势 |
| `polarization_critical` | polarization_index | > 0.8 |
| `polarization_high` | polarization_index | 0.6 ~ 0.8 |
| `silence_high` | silence_rate | > 0.5 |
| `silence_medium` | silence_rate | 0.3 ~ 0.5 |
| `response_ineffective` | 权威回应后效果 | 效果不显著 |

---

### 14. 干预窗口 (Intervention Window)

**定义**: 有效干预的时间窗口

**结构**:
```json
{
  "current_step": 5,
  "recommended_intervention_step": 8,
  "window_closing": false,
  "expected_effectiveness": 0.35
}
```

**解读**:
- `current_step`: 当前推演步数
- `recommended_intervention_step`: 建议干预的步数
- `window_closing`: 干预窗口是否正在收窄
- `expected_effectiveness`: 预期干预效果（0-1）

---

### 15. 干预策略建议 (Intervention Strategies)

**定义**: 系统推荐的具体干预措施

**策略类型**:

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| response | 官方权威回应 | 负面信念传播初期，权威可信 |
| prevent | 预防传播 | 预测负面信念将快速扩散 |
| amplify | 放大正面信念 | 正面信念已有一定传播基础 |
| depolarize | 缓解极化 | 群体严重分裂 |
| multi | 多策略组合 | 复杂舆情 |

**策略结构**:
```json
{
  "type": "response",
  "priority": 1,
  "description": "官方权威回应",
  "expected_effect": "预计降低误信率15-25%",
  "timing": "建议在3步内实施",
  "risk": "可能触发逆火效应"
}
```

## Agent 个体指标

点击网络节点可查看单个 Agent 的详细属性：

### 观点值 (Opinion)

- 范围: -1 ~ +1
- -1: 完全误信负面信念
- 0: 中立
- +1: 完全正确认知

### 信念强度 (Belief Strength)

- 范围: 0 ~ 1
- 越高表示观点越难被改变
- 受多次权威回应影响会逐渐增强

### 易感性 (Susceptibility)

- 范围: 0 ~ 1
- 越高表示越容易受他人观点影响
- 受社交网络结构影响

### 影响力 (Influence)

- 范围: 0 ~ 1
- 越高表示对他人观点影响越大
- 在无标度网络中，超级节点有极高影响力

### 恐惧孤立感 (Fear of Isolation)

- 范围: 0 ~ 1
- 越高越害怕表达与群体不同的观点
- 影响是否"沉默"

### 暴露状态 (Exposure)

| 字段 | 类型 | 说明 |
|------|------|------|
| `exposed_to_negative` | bool | 是否接触过负面信息 |
| `exposed_to_positive` | bool | 是否接触过正面信息 |

### 行动 (Action)

Agent 可能采取的行动：

| 行动 | 说明 |
|------|------|
| 转发 | 传播收到的信息 |
| 评论 | 发表个人观点 |
| 观望 | 观察但不行动 |
| 接受权威回应 | 主动接受纠正信息 |

### 情绪 (Emotion)

Agent 的情绪状态，可能值：
- 焦虑、恐惧、愤怒、困惑、理性、乐观、坚定

## 观点分布直方图

将观点值分为若干区间，统计各区间人数：

```
人数
  │
  │       ■■■
  │    ■■■■■■■■■■■
  │ ■■■■■■■■■■■■■■■■■■■■■
  │■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  └────────────────────
   -1  -0.6  -0.2  0.2  0.6   1
    │← 误信 →│    │← 正确认知 →│
```

### 颜色编码

| 颜色 | 观点范围 | 含义 |
|------|----------|------|
| 红色 | opinion < OPINION_THRESHOLD_NEGATIVE (-0.1) | 误信负面信念 |
| 橙色 | OPINION_THRESHOLD_NEGATIVE (-0.1) ≤ opinion ≤ OPINION_THRESHOLD_POSITIVE (0.1) | 中立 |
| 绿色 | opinion > OPINION_THRESHOLD_POSITIVE (0.1) | 正确认知 |

## 趋势曲线解读

### 典型曲线形态

#### 1. 负面信念快速扩散后权威回应
```
误信率:  升 → 峰值 → 降（权威回应后）
正确认知率:  低 → 升（权威回应后）
极化指数:    低 → 高 → 中
```

#### 2. 持续极化（无有效权威回应）
```
误信率:  升 → 稳定高位
正确认知率:  稳定低位
极化指数:    升 → 稳定高位
```

#### 3. 成功引导舆论
```
误信率:  升 → 峰值 → 快速下降
正确认知率:  稳步上升
极化指数:    升 → 降（趋于一致）
```

- 关注悲观场景的峰值

## 知识图谱指标

### 实体数量 (Entity Count)

**定义**: 知识图谱中提取的实体数量

**说明**: 反映事件的复杂程度，实体越多信息越丰富

### 关系数量 (Relation Count)

**定义**: 知识图谱中提取的关系数量

**说明**: 反映实体之间的关联程度

### 实体影响力分布

**定义**: 各实体对观点演化的影响力权重

**计算**: 基于实体重要性（1-5）反向映射
- 重要性 1 → 影响力 1.0
- 重要性 5 → 影响力 0.2

---

## 智库专报指标

推演结束后生成的智库专报包含以下分析：

1. **宏观趋势**: 负面信念率、正面信念率的变化趋势
2. **微观切片**: 典型 Agent 的决策过程分析（含知识图谱理解）
3. **极化分析**: 群体观点分布和演变
4. **风险回顾**: 风险预警的准确性验证
5. **干预评估**: 干预措施的效果评估
6. **政策建议**: 基于推演结果的可操作建议

---

## 监控建议

| 关注指标 | 告警阈值 | 含义 |
|----------|----------|------|
| 负面信念率 | > 70% | 误信失控 |
| 极化指数 | > 0.8 | 群体严重分裂 |
| 沉默率 | > 50% | 多数人不敢发声 |
| 公域/私域差异 | > 30% | 信息茧房严重 |
| 预测悲观值 | > 60% | 风险极高 |
| 干预窗口 | 收窄中 | 需尽快行动 |
