# 指标解释

## 概述

系统通过多个指标量化舆论演化过程，帮助研究者理解信息传播和观点演化的规律。

## 核心指标

### 1. 谣言传播率 (Rumor Spread Rate)

**定义**: 当前相信谣言的人群比例

**计算公式**:
```
rumor_spread_rate = count(opinion < -0.2) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 越高表示谣言传播越广泛
- 辟谣后应逐渐下降

**可视化**: 红色区域面积占比

---

### 2. 真相接受率 (Truth Acceptance Rate)

**定义**: 当前相信真相的人群比例

**计算公式**:
```
truth_acceptance_rate = count(opinion > 0.2) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 越高表示真相被越多的人接受
- 辟谣后应逐渐上升

**可视化**: 绿色区域面积占比

---

### 3. 平均观点 (Average Opinion)

**定义**: 群体观点的平均值

**计算公式**:
```
avg_opinion = sum(opinion_i) / N
```

**解读**:
- 范围: -1 ~ +1
- 负值: 群体倾向相信谣言
- 正值: 群体倾向相信真相
- 0: 群体观点中立

**可视化**: 观点分布直方图的"重心"

---

### 4. 极化指数 (Polarization Index)

**定义**: 群体观点的分歧程度

**计算公式**:
```
polarization = sqrt(mean(opinion_i^2))
```

或使用标准差变体:
```
polarization = std(opinion)
```

**解读**:
- 范围: 0 ~ 1
- 0: 群体观点完全一致
- 1: 群体观点两极分化（完全相信谣言 vs 完全相信真相）

**高极化场景**:
- 双方互相不信任
- 信息茧房效应强
- 缺乏共同信息基础

---

### 5. 沉默率 (Silence Rate)

**定义**: 不表达观点的人群比例

**计算公式**:
```
silence_rate = count(is_silent == True) / total_population
```

**解读**:
- 范围: 0% ~ 100%
- 反映"沉默的螺旋"效应
- 高沉默率可能导致极端观点主导舆论

---

## 双层网络指标

当启用双层网络模式时，额外提供以下指标：

### 6. 公域谣言率 (Public Rumor Rate)

**定义**: 公域网络中相信谣言的用户比例

**计算范围**: 仅公域网络中的节点

### 7. 公域真相率 (Public Truth Rate)

**定义**: 公域网络中相信真相的用户比例

### 8. 私域谣言率 (Private Rumor Rate)

**定义**: 私域社群中相信谣言的用户比例

### 9. 私域真相率 (Private Truth Rate)

**定义**: 私域社群中相信真相的用户比例

### 10. 社群数量 (Number of Communities)

**定义**: 私域网络中独立社群的数量

**说明**: 反映信息在私域中的隔离程度

### 11. 大V数量 (Number of Influencers)

**定义**: 公域网络中超级节点的数量

**说明**: 高影响力用户，可左右舆论走向

## Agent 个体指标

点击网络节点可查看单个 Agent 的详细属性：

### 观点值 (Opinion)

- 范围: -1 ~ +1
- -1: 完全相信谣言
- 0: 中立
- +1: 完全相信真相

### 信念强度 (Belief Strength)

- 范围: 0 ~ 1
- 越高表示观点越难被改变
- 受多次辟谣影响会逐渐增强

### 易感性 (Susceptibility)

- 范围: 0 ~ 1
- 越高表示越容易受他人观点影响
- 受社交网络结构影响

### 影响力 (Influence)

- 范围: 0 ~ 1
- 越高表示对他人观点影响越大
- 在无标度网络中，超级节点有极高影响力

### 恐惧孤立感 (Fear of Isolation)

- 范围: 0 ~ 1
- 越高越害怕表达与群体不同的观点
- 影响是否"沉默"

### 行动 (Action)

Agent 可能采取的行动：

| 行动 | 说明 |
|------|------|
| 转发 | 传播收到的信息 |
| 评论 | 发表个人观点 |
| 观望 | 观察但不行动 |
| 辟谣 | 主动纠正错误信息 |

### 情绪 (Emotion)

Agent 的情绪状态，可能值：
- 焦虑、恐惧、愤怒、困惑、理性、乐观、坚定

## 观点分布直方图

将观点值分为若干区间，统计各区间人数：

```
人数
  │
  │       ■■■
  │    ■■■■■■■■■■■
  │ ■■■■■■■■■■■■■■■■■■■■■
  │■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  └────────────────────
   -1  -0.6  -0.2  0.2  0.6   1
    │← 谣言 →│    │← 真相 →│
```

### 颜色编码

| 颜色 | 观点范围 | 含义 |
|------|----------|------|
| 红色 | opinion < -0.2 | 相信谣言 |
| 橙色 | -0.2 ≤ opinion ≤ 0.2 | 中立 |
| 绿色 | opinion > 0.2 | 相信真相 |

## 趋势曲线解读

### 典型曲线形态

#### 1. 谣言快速扩散后辟谣
```
谣言传播率:  升 → 峰值 → 降（辟谣后）
真相接受率:  低 → 升（辟谣后）
极化指数:    低 → 高 → 中
```

#### 2. 持续极化（无有效辟谣）
```
谣言传播率:  升 → 稳定高位
真相接受率:  稳定低位
极化指数:    升 → 稳定高位
```

#### 3. 成功引导舆论
```
谣言传播率:  升 → 峰值 → 快速下降
真相接受率:  稳步上升
极化指数:    升 → 降（趋于一致）
```

## 智库专报指标

推演结束后生成的智库专报包含以下分析：

1. **宏观趋势**: 谣言传播率、真相接受率的变化趋势
2. **微观切片**: 典型 Agent 的决策过程分析
3. **极化分析**: 群体观点分布和演变
4. **政策建议**: 基于推演结果的可操作建议

---

## 监控建议

| 关注指标 | 告警阈值 | 含义 |
|----------|----------|------|
| 谣言传播率 | > 70% | 谣言失控 |
| 极化指数 | > 0.8 | 群体严重分裂 |
| 沉默率 | > 50% | 多数人不敢发声 |
| 公域/私域差异 | > 30% | 信息茧房严重 |
